Form Checkboxes

Rhinobox

Our custom checkbox element is called rhinobox. By default, these are stacked.

Inline Rhinobox

Add the rhinobox--inline modifier to create inline checkboxes.

Form Radios

Rhinodio

Our custom radio element is called rhinodio. By default, these are stacked.

Inline Rhinodio

Add the rhinodio--inline modifier to create inline radios.

Form Selects

Rhinoselect

Our custom select element is called rhinoselect. Simply wrap a select element (containing the standard form__control class) in a container and give it the rhinoselect class.

Small Rhinoselect

Add the rhinoselect--sm modifier to the rhinoselect class.
Add the form__conrol--sm modifier to the form__control class.
