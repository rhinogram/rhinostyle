RhinoStyle: Grid

Grid

Flexbox Grid - http://flexboxgrid.com/

We have rolled our own flexbox grid system based on Flexbox Grid. This was done so that we could control gutter widths, offer various gutter sizes, and precisely target browsers of our choosing. Documentation at Flexbox Grid can still be followed.




Flexbox Grid Modifiers

Condensed Spacing

Add row--condensed modifier to row when you want tighter space between columns.
